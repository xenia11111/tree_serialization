# tree_serialization
Дерево сериализуется в виде строки : <корневой узел>[<листовой узел>, <внутренний узел>[...]] 
Например следующее дерево будет реализовано так: 8["bar"[2.015[9],2015,"2015"],"baz"["foo",6.28318["hello"]]]
.                          8
.                       /      \
.                   "bar"         "baz"
.                   /  |  \          |  \
.              2.015 2015 "2015"    "foo" 6.28318
.                  |                          |
.                  9                        "hello"
.
